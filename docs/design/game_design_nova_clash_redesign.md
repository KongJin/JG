# Game Design Redesign Archive

이 문서는 리디자인 초안 보관본이다.
내용은 `../../agent/game_design.md`로 승격되었으며, 현재 게임 방향의 SSOT는 `../../agent/game_design.md`다.

## 역할

- 새 방향으로 전환하기 전의 초안 결정 기록
- 초기 발상과 승격 시점을 설명하는 보조 문서

## 사용 규칙

- 신규 게임 규칙, MVP 범위, 게임 플로우는 이 문서에 추가하지 않는다.
- 제품 판단과 구현 기준은 `../../agent/game_design.md`를 따른다.
- 구현과 현재 코드의 차이는 별도 migration 문서에서 관리한다.
